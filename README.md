AI Career Coach

An AI-powered career development platform that helps users analyze their resume, generate a personalized career roadmap, and prepare for interviews — all powered by Google Gemini.

 Features


 Authentication — Secure registration and login with JWT (7-day expiry) and bcrypt password hashing
 Resume Analysis — Upload a resume (PDF, DOCX, or TXT) and get AI-powered feedback on skills, experience level, strengths, and areas for improvement
 Career Roadmap — Auto-generates a personalized 5–6 milestone career development plan based on your resume, with progress tracking and a "Regenerate" option
 Interview Prep — Generates 6–8 AI practice interview questions (behavioral, technical, and situational) tailored to your resume and target role
 Profile — View your account info and resume status at a glance
 Settings — Toggle between light and dark themes, manage your session
 Dashboard — A central hub with stats, skill progress, and roadmap overview


🛠️ Tech Stack

FRONTEND
Next.js 16 (App Router) + React 19 + TypeScript
Tailwind CSS v4
Radix UI + shadcn/ui components
react-hook-form + Zod for form validation
Recharts for data visualization
Lucide React for icons
next-themes for dark/light mode
Axios for API requests

BACKEND
Express.js + Node.js
MongoDB + Mongoose
Google Gemini API (Gemini 2.5 Flash) for AI features
Multer for file uploads
pdf-parse and mammoth for resume text extraction
jsonwebtoken + bcryptjs for auth
