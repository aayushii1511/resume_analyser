# SkillForge AI — AI Resume Analyzer

A full-stack MERN application that uses Google Gemini AI to analyze resumes, provide ATS compatibility scores, identify skill gaps, and generate personalized career recommendations.

---

## Tech Stack

**Frontend**
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui

**Backend**
- Node.js + Express
- MongoDB Atlas + Mongoose
- JWT Authentication
- Google Gemini AI (gemini-2.5-flash)

---

## Features

- User registration and login with JWT authentication
- Protected routes on both frontend and backend
- Resume upload (PDF, DOCX, TXT)
- AI-powered resume analysis via Gemini 2.5 Flash
- ATS compatibility score (0–100)
- Strengths and weaknesses breakdown
- Personalized career recommendations
- Dashboard with skill progress tracking and learning roadmap
